# Calc

```bash
python3 {baseDir}/scripts/calc.py "expression"
python3 {baseDir}/scripts/calc.py convert VAL FROM TO
```

Examples: `calc.py "2**10"`, `calc.py "import math; math.sqrt(144)"`, `calc.py convert 100 F C`
