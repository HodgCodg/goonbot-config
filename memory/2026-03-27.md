# 2026-03-27

## OpenClaw Context Window Limitation

**Issue:** OpenClaw enforces a hard-coded context window cap (~32k-200k tokens) regardless of the `contextWindow` value configured in `openclaw.json`, even when using models that support larger windows (e.g., Claude Opus 4.6 with 1M token context).

**Source:** GitHub issue #12644 - "Support context windows larger than 200k"

**Workaround:** Set `contextWindow` in model definition, but OpenClaw's internal compaction logic may still enforce its own limits depending on version.

```json
{
  "models": {
    "providers": {
      "lmstudio": {
        "models": [{
          "id": "your-model-id",
          "contextWindow": 196608,
          "maxTokens": 8192
        }]
      }
    }
  }
}
```

**Note:** May require OpenClaw update or source modification to fully utilize model's native context window.
